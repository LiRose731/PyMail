# PyMail
轻量无广告邮箱客户端 | 本地运行 | 支持所有邮箱

支持几乎所有主流邮箱：QQ邮箱、163邮箱、Outlook、企业邮箱等

## ✨ 特点
✅ 本地运行，不上传任何数据
✅ 无广告、无后台、无捆绑
✅ 打开即用，无需安装
✅ 支持SMTP/POP3，全邮箱兼容
✅ 界面简洁清爽，没有冗余功能

## 📦 下载使用
1.  在仓库里下载 `pymail.exe`
2.  双击打开程序，会自动启动并打开浏览器
3.  输入你的邮箱地址和授权码即可登录

## 💡 关于授权码
主流邮箱（QQ、163等）为了安全，登录第三方客户端需要使用授权码：
1.  登录你的邮箱网页版，开启「SMTP/POP3服务」
2.  在设置中找到「授权码」选项，按提示生成授权码
3.  把授权码填到PyMail的密码框里即可登录
